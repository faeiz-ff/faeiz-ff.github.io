- Simpl: Indonesian Mock Programming Language

It's a recursive acronym. Saya "menciptakan" bahasa pemrograman namanya SIMPL. Sintaksnya cukup simpl.. semua keywords masing-masing 5 huruf. You can run it here! Because it runs on JavaScript...

It barely works.. but it works!! (Hopefully you can eyeball the grammar) I wrote the lexer, parser, and interpreter from scratch, for learning purposes. Walau sepertinya akan saya kembangkan menjadi bahasa mini untuk mengajarkan pemrograman.. Who knows!
